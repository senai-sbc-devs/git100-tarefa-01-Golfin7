Git Workflow

Um guia para comandos essenciais do Git.

Clone

Copie um repositório Git para sua máquina local.

git clone

Branch

Crie e gerencie branches no Git.

git branch

Commit

Salve as mudanças no repositório.

git commit -m "Mensagem do Commit"

Merge

Combine as mudanças de diferentes branches.

git merge

Pull Request

Solicite a incorporação de mudanças ao branch principal.

git pull-request
